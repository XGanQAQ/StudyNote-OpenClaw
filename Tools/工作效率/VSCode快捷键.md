# VS Code 快捷键速查

## 基本操作

| 快捷键 | 操作 |
|--------|------|
| Ctrl + Shift + P | 命令面板 |
| Ctrl + , | 设置 |
| Ctrl + B | 切换侧边栏 |
| Ctrl + N | 新建文件 |
| Ctrl + O | 打开文件 |
| Ctrl + S | 保存 |
| Ctrl + K, S | 全部保存 |

## 编辑与格式

| 快捷键 | 操作 |
|--------|------|
| Shift + Alt + F | 格式化代码 |
| Shift + Alt + ↑/↓ | 复制行 |
| Ctrl + Shift + K | 删除行 |
| Alt + ↑/↓ | 移动行 |
| Alt + 点击 | 多光标 |

## 查找与导航

| 快捷键 | 操作 |
|--------|------|
| Ctrl + Shift + F | 全局查找 |
| Ctrl + F | 文件内查找 |
| Ctrl + H | 查找替换 |
| Ctrl + G | 跳转到行 |
| Ctrl + P | 跳转到文件 |
