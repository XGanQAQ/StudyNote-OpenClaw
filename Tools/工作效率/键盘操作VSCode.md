# VS Code 键盘操作

## 基本导航

| 快捷键 | 操作 |
|--------|------|
| Ctrl + Shift + P / F1 | 命令面板 |
| Ctrl + P | 快速打开文件 |
| Ctrl + ` | 打开终端 |
| Ctrl + B | 切换侧边栏 |

## 视图切换

| 快捷键 | 操作 |
|--------|------|
| Ctrl + Shift + E | 文件资源管理器 |
| Ctrl + Shift + F | 搜索 |
| Ctrl + Shift + G | 源代码管理（Git） |
| Ctrl + Shift + D | 调试 |
| Ctrl + Shift + X | 扩展 |

## 编辑

| 快捷键 | 操作 |
|--------|------|
| Alt + ↑/↓ | 移动当前行 |
| Alt + Shift + ↑/↓ | 复制当前行 |
| Ctrl + Shift + K | 删除当前行 |
| Ctrl + / | 注释/取消注释 |
| Shift + Alt + F | 格式化代码 |
| Ctrl + . | 快速修复 |
| F2 | 重命名符号 |

## 多光标

- `Alt + 鼠标点击`：添加光标
- `Ctrl + Alt + ↑/↓`：向上/下添加光标

## 调试

| 快捷键 | 操作 |
|--------|------|
| F5 | 启动/继续 |
| F10 | 单步跳过 |
| F11 | 单步进入 |
| Shift + F5 | 停止 |
| Shift + F11 | 跳出 |
