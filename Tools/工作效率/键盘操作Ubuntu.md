# Ubuntu 纯键盘操作

## 启动器和应用切换

| 快捷键 | 操作 |
|--------|------|
| Super（Win键） | 打开应用搜索 |
| Alt + Tab | 切换应用 |
| Alt + ` | 切换同应用的多窗口 |
| Super + E | 打开文件管理器 |
| Ctrl + Alt + T | 打开终端 |

## 窗口管理

| 快捷键 | 操作 |
|--------|------|
| Super + ↑ | 最大化 |
| Super + ↓ | 最小化 |
| Super + ←/→ | 左/右半屏 |
| Alt + F4 | 关闭窗口 |

## 桌面管理

| 快捷键 | 操作 |
|--------|------|
| Super + W | 显示所有窗口 |
| Super + S | 显示所有工作区 |
| Ctrl + Alt + ↑/↓ | 切换工作区 |
| Super + L | 锁屏 |
