# Dify 模板类型

## 基础模板

### 🗨️ 聊天助手
基于 LLM 的对话型应用，支持多轮对话。
- **适用**：客服、教育、医疗、金融

### 🤖 Agent
利用 LLM 推理能力，自主进行任务规划、拆解、工具调用。
- **适用**：问答、信息检索、任务执行

### 📝 文本生成应用
根据提示自动生成高质量文本。
- **适用**：新闻、广告、翻译、营销

## 进阶模板

### 🔄 Chatflow
面向对话类场景的工作流，支持多步逻辑处理。
- **适用**：智能客服、语义搜索

### ⚙️ 工作流（Workflow）
将复杂任务分解为多个节点（步骤），降低系统复杂度。
- **适用**：自动化翻译、数据分析、批处理
