# 幻兽帕鲁服务器搭建（Linux）

## 步骤

1. ✅ 内网穿透（FRP）
2. ✅ Linux 运行帕鲁服务端
3. ✅ 配置穿透
4. ✅ 实际测试

## FRP 配置

```ini
[[proxies]]
name = "palworld"
type = "udp"
localIP = "127.0.0.1"
localPort = 8211
remotePort = 8211
```

注意协议为 **UDP**，云服务器**记得开放防火墙端口**。

## Linux 服务端

使用 steamcmd 下载服务端，参考 [steamcmd 文档](https://developer.valvesoftware.com/wiki/Zh/SteamCMD) 和 [开服教程](https://zhuanlan.zhihu.com/p/680966903)。
