# 算术平均、几何平均、调和平均详解

## 算术平均（Arithmetic Mean）

**定义**：A = (1/n)∑xᵢ

**意义**：总量 ÷ 个数。每个样本对结果贡献相同。

**典型应用**：
- 平均成绩、平均收入
- 平均误差（MSE 的基础）
- 机器学习中损失函数取平均

**不适用**：数据极端不均衡、比例/速率问题

## 几何平均（Geometric Mean）

**定义**：G = (∏xᵢ)^(1/n) 或 log G = (1/n)∑log xᵢ

**意义**：对数空间中的算术平均。各因素为乘法叠加。

**典型应用**：
- 年化收益率
- 人口增长
- 模型相对提升倍率

**特点**：对极大值不敏感，任一值为 0 则结果为 0

## 调和平均（Harmonic Mean）

**定义**：H = n / ∑(1/xᵢ)

**意义**：倒数量算术平均的倒数。被最小值强烈支配，是"木桶效应"的数学表达。

**典型应用**：
- 平均速度、吞吐率/效率
- F1-score：F1 = 2PR/(P+R)
