# 逆矩阵、列空间与零空间

## 逆矩阵 (Inverse Matrix)

逆矩阵是**撤销**一次线性变换的操作。

- 如果矩阵 $A$ 将向量 $\vec{v}$ 变换为 $A\vec{v}$，那么逆矩阵 $A^{-1}$ 能将它**变换回来**：
  $$A^{-1}(A\vec{v}) = \vec{v}$$

- **存在条件**：矩阵的行列式 $\det(A) \neq 0$（即 [[05.行列式|行列式]] 不为零）。

> 为什么 $\det = 0$ 时没有逆矩阵？
> 行列式为 0 意味着**空间被压缩到了更低的维度**（如 3D→2D、2D→1D）。
> 压缩过程可以求解（多对一），但反过来——从低维信息恢复高维信息——是不可能的，因为信息丢失了。
> 一个输入对应多个可能的原像，所以不存在唯一的逆映射。

### 几何直观
- 逆矩阵 $A^{-1}$ 的几何意义：**把变换后的空间"倒放"回原来的位置**。
- $A^{-1}A = I$（单位矩阵），相当于"先变换、再逆变换"等于什么都没做。

## 列空间 (Column Space)

矩阵的**列向量张成的空间**，记作 $C(A)$。

- 一个 $m \times n$ 矩阵 $A$ 的列空间 = 所有列向量的线性组合
- 几何意义：**所有可能的输出向量** $A\vec{v}$ 的集合（即变换后的空间）
- 列空间是矩阵变换的**像 (image/range)**

### 列空间 vs 零空间
| 概念 | 关注点 | 意义 |
|------|--------|------|
| 列空间 | 输出向量的集合 | 哪些向量是可能的变换结果？ |
| 零空间 | 输入向量的集合 | 哪些向量被压缩到了原点？ |

---

## 秩 (Rank)

秩衡量的是**变换后的空间维度**，即列空间的维度。

- **满秩 (full rank)**：秩 = 列数（输入空间的维度没有被压缩）
- **秩的直观理解**：输出空间的维度
  - 秩 = 1 → 所有输出被压缩到一条直线上
  - 秩 = 2 → 所有输出落在一个平面上
  - 秩 = 0 → 所有输出都变成零向量（零矩阵）

> **重要关系**：
> - 秩 = 列空间的维度
> - 秩 = $n$ - 零空间的维度 (秩-零化度定理)
> - 秩越大，信息保留得越多

---

## 零空间 / 核 (Null Space / Kernel)

在变换中被压缩到**原点**的所有向量的集合，记作 $N(A)$ 或 $\ker(A)$。

$$
N(A) = \{\vec{v} \mid A\vec{v} = \vec{0}\}
$$

- 当高维空间被压缩到低维时，**一整条线（或平面）上的向量都会落在原点**。
- 零空间的维度 = 输入维度 - 秩

### 几何直观
- 零空间告诉我们：变换**丢失了多少信息**
- 零空间维度越高 → 被压缩到原点的向量越多 → 信息丢失越多
- 零空间只包含 $\vec{0}$ → 变换是一一映射的（injective/one-to-one）

---

## 四个概念的联系

```
          矩阵 A (m×n)
          /         \
         /           \
   列空间           零空间
   (所有输出)       (压缩到原点的输入)
       |                |
   维度 = 秩          维度 = n - 秩
       |                |
       +---- 秩-零化度定理 ----+
       
       rank(A) + dim(null(A)) = n
```

- **满秩方阵**：$\det \neq 0$，列空间 = 整个输出空间，零空间 = $\{\vec{0}\}$，存在逆矩阵
- **不满秩方阵**：$\det = 0$，列空间是输出空间的子空间，零空间非平凡，不存在逆矩阵
