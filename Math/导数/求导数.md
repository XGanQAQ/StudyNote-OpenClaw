# 求导

## 定义

求导是高等数学中的重要概念，表示函数在某一点上的**瞬时变化率**。

导数的定义是极限：
$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

几何意义：函数曲线在某一点上的**切线斜率**。

## 基本求导公式

| 函数 | 导数 |
|------|------|
| 常数 (c) | 0 |
| 幂函数 xⁿ | nxⁿ⁻¹ |
| 指数函数 eˣ | eˣ |
| 指数函数 aˣ | aˣ ln a |
| 对数函数 ln x | 1/x |
| sin x | cos x |
| cos x | -sin x |
| tan x | sec²x |

## 求导法则

- **常数乘法**：\[c·f(x)]' = c·f'(x)
- **加减法**：\[f(x)±g(x)]' = f'(x)±g'(x)
- **乘法法则**：\[f(x)g(x)]' = f'(x)g(x) + f(x)g'(x)
- **除法法则**：\[f(x)/g(x)]' = [f'(x)g(x) - f(x)g'(x)] / [g(x)]²
- **链式法则**：\[f(g(x))]' = f'(g(x))·g'(x)

## 示例

**乘法法则**：(x·eˣ)' = 1·eˣ + x·eˣ = eˣ(1+x)

**链式法则**：\[sin(x²+1)]' = cos(x²+1)·2x
