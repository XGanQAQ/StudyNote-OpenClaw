# 六级写作模板

## 开头段：万能引入 + 点明主题

> There is no doubt that **提出你的观点**. From my perspective, this issue deserves careful consideration.

## 主体段：原因 / 影响 / 论据

**论点句：**
- First of all, **原因/理由1**.
- In addition, **原因/理由2**.
- Moreover, **原因/理由3**.

**解释扩展句：** This is mainly because **解释原因**.

**举例支持句：** For example, **具体例子**, which clearly shows that **论点**.

## 结尾段：总结 + 呼吁

> In conclusion, **重申你的观点**. Only in this way can we achieve better development.

## 好用好词

- essential / crucial / undoubtedly

## 常用句式

### 开头句式
> In modern society, **主题** is **undoubtedly** a key component of success, **which is why** ...

### 转折连接词
- To begin with（首先）
- Furthermore（此外）

### 结尾句式
- In a nutshell（简而言之）
- To conclude
