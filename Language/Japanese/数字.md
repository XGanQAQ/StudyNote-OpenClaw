# 日语数字（一到十）

| 数字 | 平假名 | 罗马字 |
|------|--------|--------|
| 一 | いち | ichi |
| 二 | に | ni |
| 三 | さん | san |
| 四 | し / よん | shi / yon |
| 五 | ご | go |
| 六 | ろく | roku |
| 七 | しち / なな | shichi / nana |
| 八 | はち | hachi |
| 九 | きゅう / く | kyuu / ku |
| 十 | じゅう | juu |

### 注意

- **四（し）** 和 **七（しち）** 有时会避免使用，因为发音与"死"相似。通常用 **よん（yon）** 替代四，用 **なな（nana）** 替代七。
- **九** 的发音有两种，**きゅう（kyuu）** 更常用。
