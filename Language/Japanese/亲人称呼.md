# 日语亲人称呼词

## 父母

| 称呼 | 平假名 | 说明 |
|------|--------|------|
| 父 | ちち | 自己家的父亲 |
| お父さん | おとうさん | 对别人/自己父亲的亲切称呼 |
| 母 | はは | 自己家的母亲 |
| お母さん | おかあさん | 对别人/自己母亲的亲切称呼 |
| お父様 | おとうさま | 尊敬的说法，正式场合 |
| お母様 | おかあさま | 尊敬的说法，正式场合 |

|------|--------|------|
| 祖父 | そふ | 自己家的爷爷 |
| お祖父さん | おじいさん | 对别人/自己爷爷的称呼 |
| 祖母 | そぼ | 自己家的奶奶 |
| お祖母さん | おばあさん | 对别人/自己奶奶的称呼 |

## 其他亲戚

- 叔父（おじ）— 叔叔
- 叔母（おば）— 阿姨
- 従兄弟（いとこ）— 表/堂兄弟
- 従姉妹（いとこ）— 表/堂姐妹
- 夫（おっと）— 丈夫
- 妻（つま）— 妻子
- 子供（こども）— 孩子

## 规律总结

- 称呼他人家人时通常加 **お** 或 **ご** 表示尊敬
- 称呼自家人时一般不加 **お**
