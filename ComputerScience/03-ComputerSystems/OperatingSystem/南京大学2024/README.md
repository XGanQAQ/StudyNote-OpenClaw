# 南京大学 2024 操作系统（蒋炎岩）

## 课程主线
以"程序 = 状态机"为核心视角，分三大主线：

1. **并发**：多线程、同步互斥、死锁、真实并发 Bug
2. **虚拟化**：进程、地址空间、可执行文件（2025 课程覆盖）
3. **持久化**：文件系统、设备驱动（2025 课程覆盖）
