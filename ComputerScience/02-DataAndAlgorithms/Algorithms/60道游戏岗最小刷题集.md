# ⏱ 推荐刷题顺序（最优路线）

按这个顺序刷效率最高：

```
数组 → 链表 → 树 → 查找排序 → DFS → DP → 数学 → 图形
```

原因：

> 前面的题型覆盖后面的思维基础

# 📊 难度分配建议

每天刷题建议：

```
2 简单 + 1 中等
```

目标水平：

> 中等题能在20分钟内独立完成

达到这个水平，笔试基本稳。

# ⭐ 最重要的刷题原则（决定你是否能拿offer）

刷题不是比数量，而是比：

✔ 是否掌握题型模板  
✔ 是否总结规律  
✔ 是否能举一反三

> 每做完一题 → 写一句总结  
> 「这题本质考什么模型？」
