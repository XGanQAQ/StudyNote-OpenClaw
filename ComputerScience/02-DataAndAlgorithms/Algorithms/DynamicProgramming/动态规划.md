# 动态规划 (DP)

## 核心思想
- **最优子结构**：问题的最优解包含子问题的最优解
- **重叠子问题**：子问题被多次求解，需备忘
- **状态转移方程**：定义如何从子问题得到原问题

## 解题步骤
1. 确定 dp 数组含义
2. 推导状态转移方程
3. 初始化 base case
4. 确定遍历顺序
5. 举例验证

## 经典问题

| 类型 | 题目 | 状态定义 |
|------|------|---------|
| 线性 DP | 斐波那契、爬楼梯 | dp[n] = dp[n-1] + dp[n-2] |
| 路径 DP | 不同路径 | dp[i][j] = dp[i-1][j] + dp[i][j-1] |
| 分割 DP | 整数拆分 | dp[i] = max(j×(i-j), j×dp[i-j]) |
| 树形 DP | 不同二叉搜索树 | dp[n] = Σ dp[i-1] × dp[n-i] |

## 优化技巧
- 空间优化：滚动数组降维
- 记忆化搜索：自顶向下递归
