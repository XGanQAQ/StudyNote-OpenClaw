# 计算机图形学

## 3D 渲染管线

```
应用阶段(CPU) → 几何阶段(GPU) → 光栅化(GPU) → 像素着色(GPU) → 输出合并
```

### 几何阶段
- **顶点着色器**：模型→世界→视图→裁剪空间变换、光照计算
- **几何/曲面细分着色器**（可选）：生成/细分图元
- **裁剪 + 屏幕映射**：视锥体裁剪、透视除法、视口变换

### 光栅化阶段
- 将三角形图元转换为屏幕像素片段
- 插值顶点属性（颜色、法线、纹理坐标）

### 像素着色阶段
- 纹理采样、光照计算（逐像素）
- 雾化、AO 等效果

### 输出合并
- 深度测试、模板测试、混合 → 帧缓冲区

## 光照模型

### 传统光照模型
| 模型 | 特点 |
|------|------|
| [Phong](Rendering/光照模型/Phong-BlinnPhong.md) | 漫反射+高光，反射向量计算 |
| [Blinn-Phong](Rendering/光照模型/Phong-BlinnPhong.md) | 半程向量替代反射向量，更高效 |

### 基于物理渲染 (PBR)
- **核心原则**：能量守恒、菲涅尔效应、微表面模型
- **参数**：Albedo、Metallic、Roughness、Normal
- **BRDF**：双向反射分布函数，描述光线如何被反射
- [PBR 详解](Rendering/PBR（基于物理的渲染）.md)
- [BRDF 详解](BRDF-双向反射分布函数.md)

### 渲染路径
| 路径 | 原理 | 适用场景 |
|------|------|---------|
| 前向渲染 | 每个物体逐光源计算 | 光源少、半透明 |
| 延迟渲染 | G-Buffer + 逐像素光照 | 大量光源 |
