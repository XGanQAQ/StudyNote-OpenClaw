# 字段（Field）与属性（Property）的区别

在面向对象编程（如 C#）中，字段和属性是类中存储和操作数据的两种不同方式。

## 字段（Field）

字段是直接存储数据的变量，表示对象的状态。

**特点：**
- 直接访问，效率较高
- 通常设为 `private`，不直接暴露给外部
- 没有额外的逻辑封装

```csharp
public class Person
{
    public string name;   // 公共字段（不推荐）
    private int age;      // 私有字段
}
```

## 属性（Property）

属性是对字段的封装，通过 `get` 和 `set` 访问或修改数据。

**特点：**
- 可在 `get`/`set` 中添加验证或逻辑
- 可设置只读（仅 `get`）或只写（仅 `set`）
- 对外暴露接口，隐藏内部实现细节

```csharp
public class Person
{
    private int age;
    
    public int Age
    {
        get { return age; }
        set
        {
            if (value >= 0)
                age = value;
            else
                throw new ArgumentException("Age cannot be negative");
        }
    }
}
```

## 对比总结

| 特性 | 字段 | 属性 |
|------|------|------|
| 访问方式 | 直接访问 | 间接访问（get/set） |
| 逻辑封装 | 无，仅存储 | 可添加验证逻辑 |
| 可控性 | 不可控 | 安全可控 |
| 推荐使用 | 私有数据 | 对外暴露或需控制的数据 |

**建议：** 字段尽量设为 `private`，使用属性来控制数据访问，确保封装性和数据安全性。
