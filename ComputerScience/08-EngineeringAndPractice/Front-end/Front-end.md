# 前端技术

## 后端渲染 vs 前端渲染
| 方式 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| 后端渲染 | 服务器生成 HTML | SEO好、首屏快 | 交互弱 |
| 前端渲染 | SPA/CSR | 交互好、体验流畅 | SEO差、首屏慢 |

## 趋势
- **SSR (服务端渲染)**：Next.js、Nuxt.js 等框架回归
- **HTMX**：用 HTML 属性代替 JS，简化交互
- **WebAssembly**：在浏览器运行 C++/Rust
