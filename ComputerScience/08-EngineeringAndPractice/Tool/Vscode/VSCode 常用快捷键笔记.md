# 📘 VSCode 常用快捷键笔记（实用版）

## 🔹 一、基础操作（最常用）

|功能|快捷键|
|---|---|
|命令面板|`Ctrl + Shift + P`|
|快速打开文件|`Ctrl + P`|
|打开最近文件|`Ctrl + R`|
|新建文件|`Ctrl + N`|
|关闭当前文件|`Ctrl + W`|
|保存|`Ctrl + S`|
|全部保存|`Ctrl + K S`|

👉 **核心建议**：  
你90%的操作其实都可以从 `Ctrl + Shift + P` 走，这是 VSCode 的“万能入口”。

---

## 🔹 二、编辑效率（写代码核心）

|功能|快捷键|
|---|---|
|复制当前行|`Shift + Alt + ↓ / ↑`|
|删除一行|`Ctrl + Shift + K`|
|移动行|`Alt + ↑ / ↓`|
|多光标添加|`Alt + Click`|
|选中相同单词|`Ctrl + D`|
|全选相同单词|`Ctrl + Shift + L`|
|撤销|`Ctrl + Z`|
|重做|`Ctrl + Y`|

👉 **你必须掌握的三个：**

- `Ctrl + D`（变量重命名神器）
    
- `Alt + ↑↓`（重排逻辑）
    
- 多光标（写批量代码效率爆炸）
    

---

## 🔹 三、代码导航（阅读项目必备）

|功能|快捷键|
|---|---|
|跳转定义|`F12`|
|查看定义（不跳转）|`Alt + F12`|
|返回上一步|`Alt + ←`|
|前进|`Alt + →`|
|文件内搜索|`Ctrl + F`|
|全局搜索|`Ctrl + Shift + F`|
|跳转到行|`Ctrl + G`|

👉 **开发核心组合：**

```
F12 → 看代码
Alt + ← → 返回
```

---

## 🔹 四、窗口 / 面板管理（多任务关键）

|功能|快捷键|
|---|---|
|打开终端|`Ctrl + ~`|
|新建终端|`Ctrl + Shift + ~`|
|分屏编辑|`Ctrl + \`|
|切换编辑组|`Ctrl + 1 / 2 / 3`|
|关闭侧边栏|`Ctrl + B`|
|Explorer|`Ctrl + Shift + E`|
|Git 面板|`Ctrl + Shift + G`|

👉 **你的场景重点（你提到多任务 + AI）：**

- 分屏 + 多窗口 是必须的
    
- `Ctrl + \` + `Ctrl + 1/2` 非常关键
    

---

## 🔹 五、代码格式 & 重构

|功能|快捷键|
|---|---|
|自动格式化|`Shift + Alt + F`|
|重命名符号|`F2`|
|快速修复|`Ctrl + .`|
|折叠代码|`Ctrl + Shift + [`|
|展开代码|`Ctrl + Shift + ]`|

## 🔹 六、AI / 工作流增强（重点给你）

结合你现在的使用习惯，这几个特别重要：

|场景|操作|
|---|---|
|同时做多个任务|`File → New Window`|
|多AI并行|多窗口 + 分屏|
|快速切文件|`Ctrl + P`|
|快速改代码|`Ctrl + D + 多光标`|

👉 **建议你的工作流：**

```
窗口1：主项目开发
窗口2：实验 / AI生成代码
窗口3：查资料 / 文档
```
