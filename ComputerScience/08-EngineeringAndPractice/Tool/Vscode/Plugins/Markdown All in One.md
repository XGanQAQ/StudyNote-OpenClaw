Markdown All in One 是一个非常常用的 VS Code 插件，基本可以把 Markdown 的编辑体验提升到“接近写文档工具”的水平。它不是一个单一功能插件，而是一个“工具合集”。

我帮你按**实际使用场景**讲一遍，而不是简单列功能（更适合你这种做开发的人理解👇）

---

# 一、最核心：快捷键增强（写 Markdown 的核心价值）

### 1️⃣ 标题快捷键

```md
Ctrl + Alt + 1~6
```

快速生成：

```md
# 一级标题
## 二级标题
```

👉 比手敲 `#` 快很多，特别适合写博客/文档

---

### 2️⃣ 加粗 / 斜体

```md
Ctrl + B   → **加粗**
Ctrl + I   → *斜体*
```

👉 和 Word 一样的习惯，几乎无学习成本

---

### 3️⃣ 列表快捷键

```md
Ctrl + Shift + ]
Ctrl + Shift + [
```

用于：

- 增加 / 减少缩进
    
- 快速生成嵌套列表
    

---

### 4️⃣ 行移动

```md
Alt + ↑ / ↓
```

👉 调整段落顺序非常爽（写技术文档常用）

---

# 二、自动生成目录（非常关键）

### 🔥 自动生成 TOC（目录）

```md
Ctrl + Shift + P
→ Markdown All in One: Create Table of Contents
```

生成：

```md
- [一、标题](#一标题)
- [二、标题](#二标题)
```

👉 适合：

- 技术文档
    
- README
    
- 博客
    

---

### 自动更新目录

当你改标题时，它会自动更新 TOC（可配置）

👉 这个功能是“工程级文档”的关键能力

---

# 三、自动补全 & 智能输入

### 1️⃣ 自动补全列表编号

```md
1.
```

回车自动变：

```md
1.
2.
3.
```

---

### 2️⃣ 自动闭合符号

输入：

```md
**text
```

会自动变：

```md
**text**
```

👉 减少 Markdown 手误

---

# 四、快捷插入结构（写作效率神器）

### 1️⃣ 插入链接

```md
Ctrl + K
```

生成：

```md
[text](url)
```

---

### 2️⃣ 插入图片

```md
Ctrl + Alt + V
```

👉 可以直接粘贴剪贴板图片（非常好用）

---

### 3️⃣ 插入代码块

```md
Ctrl + Alt + C
```

生成：

````md
```语言
代码
````

````

---

# 五、数学公式支持（适合你）

你是做技术的，这个很关键👇

支持：
```md
$$
E = mc^2
$$
````

👉 配合 VS Code 预览，可以直接渲染公式  
（适合写算法 / 技术博客 / 游戏设计文档）

---

# 六、实时预览增强

虽然预览是 VS Code 自带的，但这个插件增强了：

```md
Ctrl + Shift + V
```

👉 分屏写文档非常舒服

---

# 七、表格格式化（隐藏神功能）

### 自动对齐表格

```md
Ctrl + Shift + P
→ Format Table
```

把：

```md
|a|b|
|1|22|
```

变成：

```md
| a | b  |
|---|----|
| 1 | 22 |
```

👉 写数据文档 / 游戏配表说明时特别好用

---

# 八、任务列表（适合做 TODO）

```md
- [ ] 未完成
- [x] 已完成
```

👉 可以当轻量任务管理工具用

---

# 九、配置能力（进阶）

你可以在 settings.json 里调：

```json
"markdown.extension.toc.levels": "2..4",
"markdown.extension.toc.slugifyMode": "github"
```

👉 控制：

- 目录层级
    
- 链接风格（GitHub兼容）
    

---

结合你的背景（游戏开发 + 技术文档），推荐你这样用👇

- TOC + 代码块 + 表格
    
- 直接发布到 GitHub / 博客
    

---

---

### ✅ 3. 写项目 README

- 自动目录
    
- 快速排版
    
- 图片粘贴
    

---

### ✅ 4. 写学习笔记

- 数学公式
    
- 结构化整理

# 总结一句话

👉 **Markdown All in One = Markdown 的“工程化工具链插件”**

它的核心价值不是“功能多”，而是：

> 👉 让你写 Markdown 像写代码一样高效
