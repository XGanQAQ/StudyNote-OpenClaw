# 一、损失函数（Loss Function）

👉 **衡量：模型“错得有多离谱”**

这是**最核心的一类函数**。

## 1️⃣ 回归任务常见损失函数

### ✅ 均方误差（MSE）

[  
L = \frac{1}{n}\sum (y - \hat y)^2  
]

- 含义：误差平方的平均
    
- 特点：
    
    - 对大误差惩罚很重
        
    - 可导、好优化
        
- 用途：线性回归、神经网络回归
    

---

### ✅ 平均绝对误差（MAE）

[  
L = \frac{1}{n}\sum |y - \hat y|  
]

- 含义：误差的绝对值
    
- 特点：
    
    - 对异常值不敏感
        
    - 在 0 处不可导
        
- 用途：鲁棒回归
    

---

## 2️⃣ 分类任务常见损失函数

### ✅ 0–1 损失（理论用）

[  
L = \mathbf{1}(y \neq \hat y)  
]

- 对了 0，错了 1
    
- **不可导**
    
- 只用于理论分析
    

---

### ✅ 对数损失（Log Loss / 交叉熵）

[  
L = -\left[y\log p + (1-y)\log(1-p)\right]  
]

- 含义：预测概率和真实分布的差距
    
- 特点：
    
    - 概率解释强
        
    - 可导
        
- 用途：逻辑回归、神经网络
    

---

### ✅ 合页损失（Hinge Loss）【SVM 专属】

[  
L = \max(0,; 1 - y f(x))  
]

- 含义：
    
    - 分得“够好” → 0
        
    - 靠太近或分错 → 线性惩罚
        
- 用途：支持向量机
    

---

# 二、激活函数（Activation Function）

👉 **引入非线性，让模型“有表达能力”**

主要出现在神经网络中。

---

### ✅ Sigmoid

[  
\sigma(x) = \frac{1}{1+e^{-x}}  
]

- 输出范围：(0,1)
    
- 用途：二分类概率
    
- 缺点：梯度消失
    

---

### ✅ ReLU（最常用）

[  
f(x) = \max(0, x)  
]

- 特点：
    
    - 计算简单
        
    - 不易梯度消失
        
- 缺点：死亡 ReLU
    

---

### ✅ Tanh

[  
\tanh(x)  
]

- 输出范围：(−1,1)
    
- 比 sigmoid 更“居中”
    

---

# 三、正则化函数（Regularization Function）

👉 **控制模型复杂度，防止过拟合**

你在 SVM 里已经用过了。

---

### ✅ L2 正则化（岭）

[  
\lambda |w|^2  
]

- 含义：惩罚权重过大
    
- 效果：让模型更“平滑”
    
- SVM、线性回归常用
    

---

### ✅ L1 正则化（Lasso）

[  
\lambda |w|_1  
]

- 含义：鼓励权重变为 0
    
- 效果：特征选择
    

---

# 四、核函数（Kernel Function）

👉 **把“非线性问题”伪装成“线性问题”**

主要出现在 SVM。

---

### ✅ 线性核

[  
K(x,z) = x^\top z  
]

- 不做映射
    
- 高维稀疏数据常用
    

---

### ✅ RBF（高斯核）

[  
K(x,z) = \exp(-\gamma |x-z|^2)  
]

- 局部影响
    
- 非线性能力强
    
- 最常用
    

---

### ✅ 多项式核

[  
K(x,z) = (x^\top z + c)^d  
]

- 引入特征组合
    

---

# 五、优化目标函数（Objective Function）

👉 **训练时真正被“最小化”的函数**

通常是：

[  
\text{目标函数} = \text{损失函数} + \text{正则项}  
]

---

### 举几个你已经见过的

- 线性回归：  
    [  
    \text{MSE} + \lambda|w|^2  
    ]
    
- SVM：  
    [  
    \frac{1}{2}|w|^2 + C\sum \xi_i  
    ]
    
- 逻辑回归：  
    [  
    \text{Log Loss} + \lambda|w|^2  
    ]
    

---

# 六、概率 / 统计相关函数（你后面一定会遇到）

### ✅ 似然函数

[  
L(\theta) = P(\text{data}|\theta)  
]

- 含义：模型参数解释数据的能力
    
- 最大似然估计（MLE）
    

---

### ✅ 对数似然

[  
\log L(\theta)  
]

- 计算稳定
    
- 常作为优化目标
    

---

# 七、一张“总表”帮你快速定位

|函数类别|解决什么问题|代表函数|
|---|---|---|
|损失函数|模型错多少|MSE / Log / Hinge|
|激活函数|非线性|ReLU / Sigmoid|
|正则化函数|防过拟合|L1 / L2|
|核函数|非线性映射|RBF|
|目标函数|实际优化|Loss + Reg|
|概率函数|统计解释|似然|
