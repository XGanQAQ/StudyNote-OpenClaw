## 回归分析

### 一元线性回归
**一元线性回归关系式的建立**
- 怎么建？
	- 假设因变量 (y) 与自变量 (x) 之间存在**线性关系**，建立**回归模型 (y = ax + b)**。根据样本数据，采用**最小二乘法**(使实际值与预测值之间的**残差平方和**最小)，从而求得**回归系数** (a) 和 (b)，得到一元线性回归方程。
- 怎么预测？
	- 在已建立的一元线性回归模型基础上，将新的自变量取值代入回归方程 (y = ax + b)，即可计算得到对应的预测值 ($\hat{y}$)，从而实现对因变量的预测。

### **多项式回归**
- 如何求解 一般的方法
多项式回归通过假设因变量与自变量之间满足多项式关系，建立回归模型  
 $$
y = a_0 + a_1x + a_2x^2 + \cdots + a_kx^k  
$$
 根据样本数据，将**多项式回归转化为多元线性回归问题**，构造设计矩阵，采用最小二乘法求解回归系数向量，从而得到多项式回归方程。

### **回归分析的效果评价**
- 平均绝对误差 （mean absolute error MAE）
	- 值越小，拟合效果越好
- 均方误差 （mean squared error MSE）
	- 值越小，拟合效果越好
	- 对最大值最小值敏感
- 均方根误差（root mean squared error RMSE）
- 决定系数（R-square, $R^2$）
	- 取值范围`[0,1]`
	- 越接近 1，拟合越好。
- 校正决定系数

## K-近邻算法

### k-近邻分类算法 KNN
k-nearest neighbor
- 原理
	- k-近邻分类算法通过计算待分类样本与训练集中所有样本之间的**距离**，选取距离**最近**的 k 个样本作为邻居，并采用**多数投票**原则，将邻居中出现次数最多的类别作为该样本的预测类别。该算法不需要显式训练模型，直接基于样本进行分类。
- 思想
	- k-近邻分类算法的核心思想是“**相似样本具有相同的类别**”。通过假设样本在**特征空间**中具有**局部相似性**，认为一个未知样本的类别可以由其在**特征空间中最接近的**若干已知样本的类别来决定。

### 距离度量

- 闵可夫斯基距离
	- 闵可夫斯基距离是一类**统一的距离度量形式**，很多常见距离都是它的特例。
- 曼哈顿距离
	- 沿着坐标轴方向移动的距离之和，像在城市街区走路。
- 欧式距离
	- 空间中两点之间的**直线距离**。
- 切比雪夫距离
	- 所有维度中，**差值最大的那一个维度**。

| 距离    | 闵可夫斯基 p        | 核心思想 | 特点关键词  |
| ----- | -------------- | ---- | ------ |
| 闵可夫斯基 | 任意 (p)         | 统一框架 | 包含多种距离 |
| 曼哈顿   | (p=1)          | 坐标轴走 | 稳健、L1  |
| 欧氏    | (p=2)          | 直线最短 | 常用、敏感  |
| 切比雪夫  | ($p\to\infty$) | 最大差值 | 最坏情况   |
### k-近邻算法的性能和特点
- 优点
	- 思想直观、原理**简单**，易于理解和实现
	- 不需要显式训练模型，**适用于小规模数据集**
		- 显式训练模型指的是：先学模型，再用模型预测
	- 对数据分布无假设，**能够处理复杂的非线性分类问题**
		- [线性分类和非线性分类](线性分类和非线性分类.md)
	- 可同时用于分类和回归任务，通用性较强
- 缺点
	- **计算复杂度高**，预测阶段需要计算与所有训练样本的距离
	- 对**噪声和异常点较敏感**，尤其在 k 较小时
	- 对**特征尺度敏感**，需要进行特征**归一化处理**
	- 当**样本维度较高**时，**距离度量效果变差**，易受“维度灾难”影响

## 决策树

### 决策树的结构与建立原理
**结构：**  
决策树由**根节点、内部节点和叶节点**组成。每个内部节点表示对某一特征的判断，每条分支表示判断结果，叶节点对应最终的类别或预测值。

**建立原理：**  
从根节点开始，**递归地选择“最优特征”对数据集进行划分**，使子节点中的样本在类别或数值上尽可能“纯”。这一过程持续进行，直到满足停止条件（如样本纯度足够高或无可分特征）。

### ID3算法
决策准则
**信息增益（Information Gain）**  
ID3 选择**信息增益最大的特征**作为划分特征。
- 核心思想：  
    通过划分，使**不确定性（信息熵）下降最多**
- 特点：
    - 偏好**取值较多的特征**
    - 只能处理**离散特征**
    - 不支持缺失值
**信息增益越大越好**

### C4.5算法
决策准则
**信息增益率（Gain Ratio）**  
C4.5 在信息增益的基础上，引入“分裂信息”进行归一化。
- 核心思想：  
    在保证信息增益较大的同时，**抑制取值数目多的特征**
- 特点：
    - 可处理**连续特征**
    - 可处理**缺失值**
    - 是对 ID3 的改进版本.
**信息增益率越大越好**

### CART 算法
决策准则
**基尼指数（Gini Index）**（分类）  
或 **平方误差最小化**（回归）
- 核心思想：  
    选择使划分后样本 **纯度最高（基尼指数最小）的特征**
- 特点：
    - 每次只进行**二叉划分**
    - 既可用于**分类树**，也可用于**回归树**
    - 结构简单，工程中使用广泛
**基尼指数越小越好**

### 决策树的特点
**优点：**
- **可解释性强**，决策过程直观
- 对**数据分布要求低**
- 能处理**非线性**问题
**缺点：**
- **容易过拟合**（需剪枝）
- 对**数据扰动较敏感**
- 单棵树**泛化能力有限**

## 贝叶斯分类器

### 朴素贝叶斯分类器

**原理**  
朴素贝叶斯分类器基于**贝叶斯定理**，通过计算在给定特征条件下各类别出现的**后验概率**来进行分类。其核心思想是：对一个样本 ($X=(x_1,x_2,\dots,x_n)$)，分别计算它属于每个类别 ($C_k$) 的概率 ($P(C_k \mid X)$)，并选择**概率最大**的类别作为预测结果。

**如何工作**  
在实际计算中，朴素贝叶斯假设**特征在给定类别条件下相互独立**，即  
$$
P(X\mid C_k)=\prod_{i=1}^{n} P(x_i\mid C_k)  
$$
训练阶段通过统计训练数据，估计先验概率 ($P(C_k)$) 和条件概率 ($P(x_i\mid C_k)$)；预测阶段对新样本分别计算各类别的 ($P(C_k)\prod_i P(x_i\mid C_k$))，比较大小并输出概率最大的类别。

**特点**
- **优点**：模型**简单**、计算**效率高**、对**小样本**和**高维数据**（如文本分类）表现良好。
- **缺点**：特征**独立性假设往往不完全成立**，可能影响分类精度。
- **适用场景**：文本分类、垃圾邮件过滤、情感分析等特征维度高但相关性可接受的任务。

## 支持向量机（SVM）

### 支持向量机的分类原理

支持向量机的核心思想是：**寻找一条能够将不同类别样本分开的“最优分隔超平面”**，并使该超平面到两类中**最近样本点的间隔最大**。  
这些距离分隔面最近的样本点称为**支持向量**，它们决定了分类边界的位置。  
在样本线性不可分的情况下，SVM 通过**核函数**将数据映射到高维特征空间，在高维空间中实现线性可分，从而完成非线性分类。

### 支持向量机的特点

- **最大间隔原则**：通过最大化分类间隔，提高模型的泛化能力
- **依赖少量关键样本**：分类结果主要由支持向量决定，而非全部样本
- **适合小样本、高维数据**：在特征维度高、样本数量较少时表现稳定
- **可处理非线性问题**：借助核函数（如高斯核、多项式核）
- **计算开销较大**：在大规模数据集上训练速度较慢
- **参数与核函数选择敏感**：模型性能依赖核函数及其参数设置

## 集成学习
[8.集成学习](8.集成学习.md)

| 对比点    | Bagging | Boosting |
| ------ | ------- | -------- |
| 训练方式   | 并行      | 串行       |
| 样本权重   | 相同      | 动态调整     |
| 主要解决   | 高方差     | 高偏差      |
| 对噪声敏感性 | 较低      | 较高       |
| 代表算法   | 随机森林    | AdaBoost |
### 集成学习的原理
- 基本思想
	- 通过组合多个“弱学习器”，构建一个性能更强、更稳定的“强学习器”。
- 组合策略有哪些
	- **投票法（Voting）**（分类）
	    - **硬投票**：选择票数最多的类别
	    - **软投票**：对各类别概率求平均，再选最大
	- **平均法（Averaging）**（回归）
	    - 对多个模型输出取算术平均或加权平均
	- **加权组合**
	    - 不同模型赋予不同权重
	    - 表现好的模型权重大
	- **学习式组合（Stacking）**
	    - 用一个新的模型学习“如何组合多个模型的输出”

### Bagging 和 随机森林

### 1️⃣ Bagging

Bagging 通过 **Bootstrap 有放回采样** 生成多个训练集，**并行训练多个基学习器**，再进行投票或平均，主要用于**降低模型方差**。

### 2️⃣ 随机森林

随机森林是在 Bagging 的基础上，**在节点划分时引入特征随机选择**，进一步降低模型相关性，提升泛化能力。

### Boosting 和 Adaboost
### 1️⃣ Boosting

Boosting 采用**串行训练**方式，后续模型重点学习前一轮**分错的样本**，主要用于**降低偏差**。

### 2️⃣ AdaBoost

AdaBoost 通过**调整样本权重和学习器权重**，对多个弱分类器进行加权组合，逐步提升分类性能。

## 降维算法

### 主成分分析
- 基本思想
	- 主成分分析通过**线性变换**将原始高维特征映射到少数几个**互不相关的主成分**上，在**最大化数据方差**的同时实现降维。

- 求解过程
	- 对原始数据进行**中心化**
	- 计算样本的**协方差矩阵**
	- 对协方差矩阵进行**特征值分解**
	- 选取**最大特征值对应的特征向量**作为主成分
	- 将数据投影到选定的主成分上完成降维

### 主成分分析的特点
- **无监督**降维方法，不依赖类别标签
- 主成分之间**相互正交、互不相关**
- 能在保留主要信息的同时**降低维度与噪声**
- 属于**线性降维方法**，难以处理强非线性结构
- 主成分的物理意义通常不直观

## 聚类
### 基于划分的聚类方法

**k-均值算法**
给定簇数 _k_，通过不断“**分配样本到最近中心 → 更新簇中心**”的迭代过程，使簇内样本尽量相似、簇间差异尽量大，最终最小化簇内平方误差。

### 基于层次的聚类方法

**簇减距离度量**
用于衡量两个簇之间的相似度，常见方式有：
- **最短距离（单链接）**：取两簇中最近样本的距离
- **最长距离（全链接）**：取两簇中最远样本的距离
- **平均距离**：两簇所有样本两两距离的平均值

### 聚类评估

**兰德指数**
衡量聚类结果与真实标签的一致性，取值 `[0,1]`，越大表示聚类越准确。
**Jaccard指数**
只关注“被同时分到同一簇”的样本对，相比兰德指数对错误聚类更敏感，值越大效果越好。
