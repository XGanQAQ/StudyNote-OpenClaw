# 聚类

在**没有标签**的情况下，根据样本之间的**相似度**，把数据自动分成若干组（簇）。
## 基于划分的聚类方法

### 思想

- **预先指定簇的个数 K**
    
- 通过优化目标函数，把样本划分到 K 个簇中

### k-均值算法 ⭐

用 **K 个“中心点”** 表示 K 个簇，让每个样本分到**最近的中心**，并不断更新中心，使**簇内平方误差最小**。

#### ✅ 优点
- 简单、直观
- 速度快，适合大数据
- 易实现

#### ❌ 缺点
- **K 需要提前指定**
- 对初始中心敏感（容易陷入局部最优）
- 只能发现**“近似球形”的簇**
- 对异常点（噪声）敏感

k-means 是一种基于划分的聚类方法，其核心思想是通过**最小化簇内平方误差**，将样本划分为 K 个簇。算法通过迭代执行样本分配和中心更新两个步骤，最终收敛。其优点是实现简单、效率高，但需要预先指定 K，对初始值和异常点敏感。

## 基于层次的聚类方法

不直接给 K，而是构造一个**层次结构（树状）**，再从中“切”出聚类结果。

### 两种策略

### 1️⃣ 凝聚式（自底向上）⭐
- 每个样本先是一个簇
- 不断**合并最近的两个簇**
- 直到只剩一个簇

👉 **更常用**

#### 2️⃣ 分裂式（自顶向下）
- 所有样本一个簇
- 不断拆分
### 簇间距离的定义（重点）

- **单链接（Single Link）**  
    最近两个点的距离（容易“拉长”）
- **全链接（Complete Link）**  
    最远两个点的距离（更紧凑）
- **平均链接（Average Link）**
- 质心法

| 方法   | 簇间距离怎么算 | 直观理解   | 典型问题    |
| ---- | ------- | ------ | ------- |
| 单链接  | 最小点距    | 最近两个点  | 链式效应    |
| 全链接  | 最大点距    | 最远两个点  | 对离群点敏感  |
| 平均链接 | 点距平均    | 整体平均距离 | 计算略复杂   |
| 质心法  | 质心距离    | 中心到中心  | 距离可能不单调 |
    
📌 不同定义 → 不同聚类结果

### 特点总结

#### ✅ 优点
- 不需要事先指定 K
- 能生成**树状结构（dendrogram）**
- 结果可解释性强
    
#### ❌ 缺点
- 计算复杂度高（不适合大数据）
- 一旦合并/分裂不可撤销
- 对噪声敏感

## 基于密度的聚类方法

**簇是高密度区域**，低密度区域是噪声

### DBSCAN

根据“密度连续性”自动形成聚类，并区分噪声点

| 维度      | 归类    |
| ------- | ----- |
| 学习方式    | 无监督学习 |
| 聚类方法    | 基于密度  |
| 是否需要 K  | 否     |
| 是否能处理噪声 | 是     |
| 簇形状     | 任意    |

## 聚类评估

### 1️⃣ 内部指标（没有真实标签）

#### ⭐ 轮廓系数（Silhouette Coefficient）

$$
s = \frac{b - a}{\max(a, b)}  
$$

- a：样本到**本簇内**其他点的平均距离
    
- b：样本到**最近其他簇**的平均距离
    

📌 取值范围：**[-1, 1]**

- 越接近 1 → 聚类越好
    
#### 其他常见

- 簇内平方和（SSE）
    
- Davies–Bouldin 指数
    

### 2️⃣ 外部指标（有真实标签）

- 纯度（Purity）
    
- Rand Index
    
- Adjusted Rand Index（ARI）
