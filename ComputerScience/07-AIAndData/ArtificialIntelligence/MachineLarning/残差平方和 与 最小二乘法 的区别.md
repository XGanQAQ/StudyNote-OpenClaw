| 对比点  | 残差平方和                    | 最小二乘法                         |
| ---- | ------------------------ | ----------------------------- |
| 是什么  | 一个**数值指标 / 函数**          | 一种**方法 / 原则**                 |
| 作用   | 衡量误差大小                   | 用来求参数                         |
| 数学形式 | ($\sum (y - \hat{y})^2$) | ($\min \sum (y - \hat{y})^2$) |
| 地位   | 目标函数                     | 优化方法                          |

残差平方和是需要被最小化的**目标函数**，而最小二乘法是实现该最小化的**求解方法**。

在线性回归中，构造**残差平方和**作为目标函数。利用**最小二乘法**，对残差平方和关于回归系数求偏导，并令**偏导数为零**，得到正规方程。解该方程可得到回归系数，此时残差平方和达到最小，因此所得回归系数为最优回归系数。

### 为什么“偏导 = 0”就是最小值？

数学直觉
- 残差平方和是一个**关于回归系数的二次函数**
- 二次函数的图像是 **开口向上的抛物面**

### 梯度递减 和 最小二乘法的区别

|对比点|最小二乘法|梯度递减|
|---|---|---|
|是什么|**准则 / 目标**|**算法 / 方法**|
|解决的问题|最小化什么|怎么最小化|
|是否限定损失函数|是（平方误差）|否（任何可导函数）|
|是否迭代|❌ 不一定|✅ 一定|
|是否有解析解|常有|一般没有|
|常见应用|线性回归|线性/逻辑回归、神经网络|
- **最小二乘法 + 偏导 = 正规方程**
- **最小二乘法 + 梯度递减**
