## Machine Learning (机器学习)

### 算法分类
| 类型 | 说明 | 常见算法 |
|------|------|---------|
| 监督学习 | 带标签数据训练 | 线性回归、决策树、SVM、KNN |
| 无监督学习 | 无标签数据 | K-Means、PCA、DBSCAN |
| 半监督学习 | 少量标签+大量未标签 | 自训练、图方法 |
| 强化学习 | 环境交互奖励 | Q-Learning、DQN、PPO |

### 经典算法
| 算法 | 用途 | 特点 |
|------|------|------|
| 线性回归 | 连续值预测 | 最小二乘法、梯度下降 |
| 逻辑回归 | 二分类 | Sigmoid、交叉熵损失 |
| KNN | 分类/回归 | 基于距离、懒学习 |
| 决策树 | 分类/回归 | 信息增益、剪枝 |
| 贝叶斯分类器 | 分类 | 先验概率+条件概率 |
| SVM | 分类 | 最大间隔、核技巧 |
| K-Means | 聚类 | 无监督、迭代中心 |

### 模型评估
- 准确率/精确率/召回率/F1
- ROC/AUC
- 交叉验证
- 过拟合 vs 欠拟合
- 偏差-方差权衡
