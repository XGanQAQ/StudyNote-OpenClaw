
## Agent 导表工具
Unity + Excel + Luban 工作流，拆成一个完整的 Agent 设计

在没有AI参与下，如果我要创建一个新的数据表，需要进行如下操作
- 在GameApp的table文件夹下面新建一个excel表格
- 按照规范，定义设计你需要的字段，表名，字段类型
	- 完成表格数据的填写
	- 如果需要定义枚举类型，需要采用特定的方法
- 然后再执行导表工具，将Excel表格转换为Json格式，并生成对应的容器类

如果有AI的参与，我就可以不用严格的重复这些流程，而是发送这样的一段文字
```txt
我要创建一个名为ItemConfig的道具表格，用来存储游戏的道具数据。
道具数据中有如下字段
ID Name Cost Value
```
然后他就可以创建对应的表格，字段，并执行导出表格的任务。
