# C++ 智能指针 vs C# 垃圾回收

## 核心区别

| 维度 | C++ 智能指针 | C# GC |
|------|-------------|-------|
| 管理方式 | RAII（确定性） | 自动垃圾回收（非确定性） |
| 释放时机 | 对象离开作用域时（可预测） | GC 周期性检查（不可预测） |
| 运行时开销 | 极低（unique_ptr无额外开销） | GC 停顿 |
| 安全性 | 需开发者注意循环引用等问题 | 极大降低内存泄漏风险 |
| 非内存资源 | 自定义删除器 | IDisposable |

## C++ 智能指针类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| `unique_ptr` | 独占所有权，不可拷贝 | 拥有唯一所有权 |
| `shared_ptr` | 共享所有权，引用计数 | 多对象共享资源 |
| `weak_ptr` | 非拥有型，解决循环引用 | 打破 shared_ptr 循环 |
