# C++ 虚函数

## 概念
虚函数通过 **虚函数表（vtable）** 和 **虚指针（vptr）** 实现运行时多态。

## 关键点

### 虚函数表 (vtable)
- 每个有虚函数的类有一个虚函数表（静态数组）
- 存储该类所有虚函数的地址
- 派生类重写时替换对应位置为派生类版本

### 虚指针 (vptr)
- 每个对象有一个隐藏的 vptr，指向所属类的 vtable
- 构造函数初始化 vptr

### 动态绑定流程
1. 通过基类指针/引用调用虚函数
2. 通过对象 vptr 找到 vtable
3. 根据函数索引调用实际函数

## 注意事项
- **虚析构函数**：基类有虚函数时，析构函数必须为 virtual
- **纯虚函数**：`virtual void func() = 0;` → 抽象类不能实例化
- **性能**：间接调用有微小开销
- **对象切片**：派生类赋值给基类对象会丢失派生部分
