# C++ 高频操作与笔试常用方法

## 输入输出
```cpp
ios::sync_with_stdio(false);

```

## STL 容器速查

| 容器 | 特点 | 常用操作 |
|------|------|---------|
| vector | 动态数组 | push_back, pop_back, size, sort |
| map | 红黑树 (O(log n)) 有序 | [], find, count, erase |
| unordered_map | 哈希表 (O(1)) 无序 | [], find, count |
| set | 有序集合 | insert, count, erase |
| queue | 队列 | push, pop, front |
| stack | 栈 | push, pop, top |

## 常用算法

```cpp
#include <algorithm>
sort(v.begin(), v.end());                       // 排序
reverse(v.begin(), v.end());                    // 翻转
lower_bound(v.begin(), v.end(), x);             // 第一个 >= x
upper_bound(v.begin(), v.end(), x);             // 第一个 > x
binary_search(v.begin(), v.end(), x);           // 二分查找
accumulate(v.begin(), v.end(), 0);              // 求和
```

## 结构体排序
```cpp
struct Node { int id, score; };
bool cmp(const Node& a, const Node& b) { return a.score > b.score; }
sort(v.begin(), v.end(), cmp);
```
