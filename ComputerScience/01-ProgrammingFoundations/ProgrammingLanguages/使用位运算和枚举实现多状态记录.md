# 使用位运算和枚举实现多状态记录

通过将每个状态表示为二进制位，利用位运算高效管理状态。

## 关键操作

| 操作 | 符号 | 作用 | 示例 |
|------|------|------|------|
| 按位或 | `|` | **添加**状态 | `state \| FEATURE_A` |
| 按位与 | `&` | **检查**状态 | `state & FEATURE_A == FEATURE_A` |
| 按位非+与 | `& ~` | **移除**状态 | `state & ~FEATURE_A` |

## 示例

```csharp
[Flags]
enum PlayerState
{
    None     = 0,
    Moving   = 1 << 0,
    Jumping  = 1 << 1,
    Crouching = 1 << 2,
    Attacking = 1 << 3
}

PlayerState state = PlayerState.None;
state |= PlayerState.Moving;   // 添加移动状态
if ((state & PlayerState.Jumping) != 0) { /* 正在跳跃 */ }
state &= ~PlayerState.Moving;  // 移除移动状态
```
