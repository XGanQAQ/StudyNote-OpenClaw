# 结构体 (struct) 与类 (class) 的区别

## 核心：值类型 vs 引用类型

| 维度 | 结构体 (Struct) | 类 (Class) |
|------|----------------|------------|
| 类型 | 值类型 | 引用类型 |
| 内存 | 栈上（或内联） | 堆上 |
| 拷贝 | 值拷贝（独立副本） | 引用拷贝（同一对象） |
| 继承 | ❌ 不支持 | ✅ 支持 |
| 析构函数 | ❌ 无 | ✅ 有 |
| 适用场景 | 小型简单数据 | 复杂对象、需共享 |

## 选择指南

### 用 struct 当：
- 表示单一小值（点、颜色、坐标）
- 生命周期短
- 常作为参数传递
- 不需要继承

### 用 class 当：
- 复杂对象
- 需要继承/多态
- 数据较大、需引用传递
