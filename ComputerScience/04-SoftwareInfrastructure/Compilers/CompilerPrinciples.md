# 编译原理

## 编译器架构
```
源程序 → [词法分析] → [语法分析] → [语义分析] → [中间代码生成] → [优化] → [目标代码生成]
```

## 阶段概述

| 阶段 | 输入 | 输出 | 说明 |
|------|------|------|------|
| 词法分析 | 字符流 | Token 流 | 正则+有限自动机 |
| 语法分析 | Token 流 | 语法树 | 上下文无关文法 (BNF) |
| 语义分析 | 语法树 | 带注释的语法树 | 类型检查、符号表 |
| 中间代码生成 | 语法树 | 三地址码/IR | 与机器无关的中间表示 |
| 优化 | IR | 优化后的 IR | 常数折叠、死代码消除 |
| 目标代码生成 | IR | 目标机器码 | 寄存器分配、指令选择 |

## 关键概念
- **LL(1) / LR(1) 文法**：自顶向下/自底向上分析
- **AST**：抽象语法树
- **IR**：中间表示（如 LLVM IR）
- **汇编基础**：寄存器、指令集、寻址模式

## 汇编基础
- x86/ARM 指令集架构
- 寄存器、栈帧、调用约定
- 汇编器：将汇编代码转为机器码
- 链接器：将目标文件链接为可执行文件
