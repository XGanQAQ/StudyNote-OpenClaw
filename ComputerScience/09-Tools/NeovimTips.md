# Neovim 快捷键速查

## 基础

| 操作 | 说明 |
|------|------|
| `i` | 进入插入模式 |
| `Esc` | 返回普通模式 |
| `:` | 进入命令模式 |

## 移动

| 操作 | 说明 |
|------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w/b` | 下一个/上一个单词 |
| `0/$` | 行首/行尾 |
| `gg/G` | 文件开头/结尾 |

## 编辑

| 操作 | 说明 |
|------|------|
| `dd` | 删除一行 |
| `yy` | 复制一行 |
| `p` | 粘贴 |
| `x` | 删除字符 |
| `u` | 撤销 |

## 组合操作

| 操作 | 说明 |
|------|------|
| `dw` | 删除单词 |
| `ciw` | 修改单词 |
| `di(` | 删除括号内容 |
| `ci"` | 修改引号内容 |
| `d$` | 删除到行尾 |

## 搜索

| 操作 | 说明 |
|------|------|
| `/xxx` | 搜索内容 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |

## 行操作

| 操作 | 说明 |
|------|------|
| `o` | 下方新建一行 |
| `O` | 上方新建一行 |
| `A` | 行尾插入 |
| `I` | 行首插入 |

## 文件

| 操作 | 说明 |
|------|------|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 强制退出 |

## 缓冲区

| 操作 | 说明 |
|------|------|
| `:bn` | 下一个文件 |
| `:bp` | 上一个文件 |
