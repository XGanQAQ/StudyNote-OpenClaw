## 常用的纹理类型及其作用

这里列举了一些最常用且重要的纹理类型：

### 1. 漫反射贴图 (Diffuse Map / Albedo Map)

- **作用：** 这是最基础的颜色贴图，它定义了物体表面的固有颜色信息，即在没有光照影响下，物体本身呈现的颜色。想象一下一个没有任何光影效果的卡通版物体，它的颜色就来自漫反射贴图。
    
- **特点：** 通常不包含任何光照或阴影信息，只是纯粹的颜色。在基于物理渲染（PBR）的工作流程中，它被称为 **Albedo Map (反照率贴图)**，强调的是物体表面反射光的颜色和强度，不应包含环境光遮蔽（AO）或阴影信息。
    

### 2. 法线贴图 (Normal Map)

- **作用：** 法线贴图用于模拟物体表面的高低不平的细节，而无需增加实际的模型多边形数量。它通过存储每个像素的法线方向信息，来欺骗光线，让平面看起来有凹凸感。
    
- **特点：** 通常呈现为蓝紫色调。当光线照射到模型表面时，引擎会根据法线贴图中的信息调整光照计算，从而产生视觉上的凹凸效果。这对于在低多边形模型上表现高细节（如石头表面的纹理、皮肤毛孔、服装褶皱等）至关重要。
    

### 3. 高光贴图 (Specular Map / Metallic Map)

在传统的非PBR渲染中，有 **高光贴图 (Specular Map)**；而在现代的PBR渲染中，则主要使用 **金属度贴图 (Metallic Map)** 和 **粗糙度贴图 (Roughness Map)** 来控制高光。

- **高光贴图 (Specular Map - 传统)：** 决定了物体表面反射高光的颜色和强度。白色区域反射更多高光，黑色区域反射较少或不反射。它用于模拟塑料、旧金属等传统材质的高光。
    
- **金属度贴图 (Metallic Map - PBR)：** 定义了物体表面是金属还是非金属。白色表示金属（完全反射光线，没有固有色），黑色表示非金属（具有固有色，反射高光）。中间的灰度值表示半金属。结合粗糙度贴图，可以精确模拟各种金属和非金属材质。
    
- **粗糙度贴图 (Roughness Map - PBR)：** 定义了物体表面的光滑或粗糙程度。白色通常表示非常粗糙（光线散射，高光模糊），黑色表示非常光滑（光线集中，高光锐利）。它与金属度贴图共同决定了物体表面的反射效果。
    

### 4. 环境光遮蔽贴图 (Ambient Occlusion Map / AO Map)

- **作用：** AO贴图模拟了物体在环境光照射下，模型自身凹陷、裂缝或相邻表面之间因光线被遮挡而产生的柔和阴影效果。它能增加模型的体积感和真实感，让模型看起来不那么“平”。
    
- **特点：** 通常是灰度图，黑色区域表示光线被遮挡得更厉害，白色区域表示光线可以自由到达。AO贴图通常会在引擎中与其他贴图混合，以增强阴影细节。
    

### 5. 自发光贴图 (Emissive Map)

- **作用：** 用于表现物体自身发光的效果，例如灯光、霓虹灯、屏幕、发光的符文等。它让物体看起来像是一个光源，即使场景中没有额外的灯光照射。
    
- **特点：** 通常包含颜色信息，表示物体发光的颜色。在游戏引擎中，它会让相应区域发出光芒，有时还会影响周围环境的光照（如果引擎支持）。
    

### 6. 不透明度贴图 (Opacity Map / Alpha Map)

- **作用：** 控制物体表面的透明度。通过这张贴图，可以定义模型上哪些区域是完全透明、完全不透明或半透明的。
    
- **特点：** 通常是灰度图或带有Alpha通道的图片。黑色表示完全透明，白色表示完全不透明，灰度值表示半透明。常用于制作树叶、栅栏、玻璃、粒子特效等。
    

### 7. 高度贴图 (Height Map / Displacement Map)

- **作用：** 用于表现物体表面的真实物理凹凸，与法线贴图模拟的视觉凹凸不同，高度贴图在渲染时会实际地改变模型的几何形状。
    
- **特点：** 灰度图，白色代表高，黑色代表低。高度贴图通常需要额外的着色器（如Tessellation着色器）支持才能实现位移效果，这会增加模型的顶点数量，因此在性能要求高的游戏中不如法线贴图常用。
