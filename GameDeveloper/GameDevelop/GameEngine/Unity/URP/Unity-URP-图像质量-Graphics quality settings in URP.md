URP设置图像质量的方法

- [Unity - Manual: Universal Render Pipeline asset](https://docs.unity3d.com/6000.1/Documentation/Manual/urp/urp-asset-and-renderer.html)
	- multiple URP assets and switch between them. For example, you can have one with Shadows on and one with Shadows off
	- 但是你不能在HDRP SRP URP 之间切换
- [Unity - Manual: Display advanced properties in a URP Asset](https://docs.unity3d.com/6000.1/Documentation/Manual/urp/urp-asset-additional-settings.html)
	- Unity does not show certain advanced properties in the URP Asset by default
- [Unity - Manual: Change the active URP Asset at runtime](https://docs.unity3d.com/6000.1/Documentation/Manual/urp/quality/quality-settings-through-code.html)
- [Unity - Manual: Change URP Asset settings at runtime](https://docs.unity3d.com/6000.1/Documentation/Manual/urp/quality/change-urp-asset-settings.html)
- [Unity - Manual: Configure settings with the URP Config package](https://docs.unity3d.com/6000.1/Documentation/Manual/urp/URP-Config-Package.html)
	- URP Config 包提供了一种精细控制 URP 某些渲染设置的方法，但你需要手动管理其嵌入和更新，以确保你的项目始终使用期望的配置。
