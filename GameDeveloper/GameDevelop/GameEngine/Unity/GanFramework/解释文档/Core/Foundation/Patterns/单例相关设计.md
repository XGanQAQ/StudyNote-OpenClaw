- 继承Mono
	- SceneSingleton
		- 不会跨场景
	- GlobalMonoSingleton
		- 跨场景存在
- 纯C#
	- 未命名
		- 一开始便初始化


纯C#类通过继承特定的接口/标签，在框架核心处被注册初始化


## 适用情况

| 问题               | 是          | 否     |
| ---------------- | ---------- | ----- |
| 需要 Inspector 配置？ | Mono       | 纯 C#  |
| 需要跨场景存在？         | GlobalMono | Scene |
| 是核心逻辑 / 规则 / 数据？ | 纯 C#       | Mono  |
| 和 Unity API 强耦合？ | Mono       | 纯 C#  |

| 维度        | SceneSingleton | GlobalMonoSingleton | 纯 C# Service |
| --------- | -------------- | ------------------- | ------------ |
| 生命周期      | Scene          | 全游戏                 | App          |
| Inspector | ✔              | ✔                   | ❌            |
| Unity API | 强              | 强                   | ❌            |
| 可测试性      | 低              | 低                   | 高            |
| 数据驱动      | 一般             | 一般                  | ⭐⭐⭐          |
| 推荐比例      | 少              | 少                   | **多**        |
