## 一句话核心区分

> **Utils 是“被代码调用的工具”  
> Tools 是“被人操作的工具”**

Utils 是给程序用的刀  
Tools 是给开发者用的工具台

## Utils vs Tools 对照表

| 维度               | Utils     | Tools |
| ---------------- | --------- | ----- |
| 谁在用              | 程序        | 人     |
| 使用时机             | 运行时 / 编辑时 | 仅编辑时  |
| 是否进 Player       | 是         | 否     |
| 是否依赖 UnityEditor | 否         | 是     |
| 是否允许被剥离          | 不允许       | 必须允许  |
| 是否影响游戏逻辑         | 可能        | 不应该   |
