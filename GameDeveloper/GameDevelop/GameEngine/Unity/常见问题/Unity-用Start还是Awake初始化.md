# ⭐ 多年 Unity 架构实践最佳实践

下面是一个业界通用规则：

| 情况                       | 放 Awake | 放 Start |
| ------------------------ | ------- | ------- |
| 初始化自身字段                  | ✅       |         |
| 获取子组件 / 本对象组件            | ✅       |         |
| 创建对象内部状态                 | ✅       |         |
| 访问 ManagerLocator        |         | ✅       |
| 访问 GameManager/UIManager |         | ✅       |
| 注册事件（依赖外部系统）             |         | ✅       |
| Mirror/Netcode 相关联机初始化   |         | ✅       |
| UI 初始化（按钮绑定、读取系统状态）      |         | ✅       |
