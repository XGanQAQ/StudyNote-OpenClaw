# StudyNote-OpenClaw

📚 由 AI 整理的学习笔记（公开版）

本仓库由 [OpenClaw](https://github.com/openclaw/openclaw) 自动维护，内容来自我的私有笔记仓库 [Obsidian-StudyNote](https://gitea.ganx.top/xgan/obsidian-StudyNote)。

## 说明

- **来源**：我的私有 Obsidian 笔记
- **处理者**：OpenClaw AI 助手
- **筛选规则**：跳过私人内容（`Private/`）、AI 自身笔记（`OpenClaw/`）和脚本文件（`Scripts/`）
- **同步频率**：每日自动

## 目录结构

目录结构与原笔记仓库保持一致，涵盖但不限于以下分类：

| 目录 | 内容 |
|------|------|
| `ComputerScience/` | 计算机科学（网络、系统、AI、工具等） |
| `Math/` | 数学（线性代数、统计学、微积分等） |
| `Language/` | 语言学习（英语、日语等） |
| `Art/` | 美术设计（像素画、绘画风格等） |
| `GameDeveloper/` | 游戏开发相关 |
| `Notes/` | 读书笔记、经验总结 |
| `Tools/` | 工具使用笔记 |
